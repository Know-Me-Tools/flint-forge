01

Brand Philosophy

KnowMe sits where personal intelligence meets architectural precision. The brand is warm but never casual, personal but never sentimental. The ember isn't decoration — it's conviction.

Brand Pillars
#PillarExpression
01Deeply PersonalAI that knows you by name, context, history. Not a generic assistant — a calibrated model of you.
02Ember ConvictionOne hot accent against cool space. Restraint makes the ember more powerful, not less.
03Architectural ClarityNothing accidental. Spacing, type, layout — each decision is load-bearing, not decorative.
04Quiet IntelligenceSophisticated, not clever. Depth over breadth. Precision over volume.

Core Promise: KnowMe doesn't feel like software. It feels like a system that was built for exactly you — because it was.

Logo Rules
✓ CORRECT USAGE
Use charcoal body + ember dot on light backgrounds
Use off-white body + bright ember dot on dark backgrounds
Minimum clear space = 1× icon width on all sides
Minimum size: 16px for favicon, 24px for nav, 52px for hero
Monochrome version for single-color print contexts
✗ NEVER DO
Do not recolor the ember dot to any non-brand color
Do not stretch or distort the mark proportions
Do not use drop shadows or outer glows
Do not place on low-contrast backgrounds
Do not use the ember accent version in UI chrome (nav, footer)

10

Motion Principles

Motion communicates precision, not energy. Transitions are fast and decisive. Nothing bounces. Nothing lingers. The interface moves like it was designed.

Duration: 150ms interactions · 250ms page transitions · 350ms overlays
Easing: ease · ease-out · cubic-bezier(0.4,0,0.2,1)
Hover: opacity 0.88 + translateY(-1px) on CTAs
Mode toggle: background + color 250ms ease
No: spring physics · bounce · elastic · scale >1.02
